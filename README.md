Система фотореалистичного рендера облаков в реальном времени в Unity HDRP, основанная на вычислительных шейдерах, RayMarching и объемном рендеринге

<img width="1276" height="604" alt="силвер лайнинг" src="https://github.com/user-attachments/assets/66b7b222-e60d-42c0-9405-b2e157a79e28" />

Облака процедурно генерируются с помощью заранее сгенерированных шумов Perlin, Worley и Perlin-Worley, записанных в трехмерные и двухмерные текстуры. Такой подход позволяет быстро и эффективно высчитывать многокилометровые пространства неба, не тратя слишком много ресурсов в рамках вычислений в реальном времени. Работа опирается на подходы, применяемые Guerrilla Games (Horizon: Zero Dawn / Forbidden West) и Rockstar Games (Red Dead Redemption II), с некоторыми личными доработками и решениями.
